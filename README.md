## Installation 🛠️

Clone the repository:
   ```sh
   git clone https://github.com/valentyn-protsyuk/RR-Playwright.git
   ```
## Navigate to the project directory:



  ```sh
cd RR-Playwright
 ```
## Install dependencies:

  ```sh

npm install
 ```

## Running Tests 🚀

  ```sh

npx playwright test
 ```
